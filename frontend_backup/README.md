# Frontend

This folder contains the frontend app copied from `https://github.com/geeta-2707/react-transform-wiz.git`.

## Setup

From the `frontend` directory, install dependencies:

```bash
npm install
```

> If you prefer Bun and have it installed, you can also use `bun install`, but this README uses `npm` because the project includes `package-lock.json`.

## Available commands

Run these from `frontend`:

```bash
npm run dev
```

Start the app in development mode.

```bash
npm run build
```

Build the production app.

```bash
npm run build:dev
```





## Environment

This project uses environment variables from `.env`.

If you need to update Supabase values, edit `.env` or create a `.env.local` file with the same variable names.
